Prazer, eu sou Rebeca! Desenvolvedora Fullstack.

Sou desenvolvedora fullstack júnior em formação, movida pela curiosidade e pela vontade constante de aprender. Durante minha trajetória no curso técnico em Desenvolvimento de Sistemas, venho criando projetos que unem lógica de programação, banco de dados e desenvolvimento web, explorando como a tecnologia pode gerar soluções práticas e inovadoras para o dia a dia. Tenho experiência com HTML, CSS, JavaScript, Java e MySQL, além de utilizar ferramentas como Git e VS Code no meu processo de desenvolvimento. Meu objetivo é continuar evoluindo como profissional e contribuir para projetos que façam a diferença por meio da tecnologia. Outros cursos que ja concluí na instituição do SENAI foram Informática Básica e Excel Avançado, tendo assim conhecimento em atividades administrativas e planilhas.

    EletroniX

    Este projeto foi desenvolvido em sala de aula
    como parte do curso Técnico em
    Desenvolvimento de Sistemas do SENAI.
    O objetivo principal foi criar a tela inicial de um catálogo de produtos para uma loja online, inspirado no modelo do site Mercado Livre.
    A aplicação apresenta uma interface simples, funcional e visualmente agradável, permitindo ao usuário navegar por categorias e visualizar os produtos disponíveis.

    API Demon Slayer

    Este projeto está sendo desenvolvido em sala de aula como parte do curso Técnico em Desenvolvimento de Sistemas do SENAI. O objetivo principal é consumir uma API pública sobre o anime Demon Slayer e criar uma tela gráfica, exibindo informações como sinopse, personagens, estilos de combate e etc. Projeto desenvolvido com HTML, CSS e JavaScript.
